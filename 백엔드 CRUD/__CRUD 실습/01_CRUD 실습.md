### 01_CRUD 실습

#### ◾ 벡엔드와의 대화 

* 백엔드에 정보를 보내는 3가지 방법 : path, query, req body
* 실제 우리 프로젝트에 응답을 처리 



#### ◾ 벡엔드의 CRUD

> 데이터를 처리하는 4가지 기본 작업 

* Create : 새로운 데이터 생성 추가 

* Read : 데이터 조회 및 검색

* Update : 기존 데이터 변경 또는 갱신 

* Delete : 데이터 삭제
