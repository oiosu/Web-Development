## 1-11. css 프레임워크 

* #### Bootstrap, tailwind css, mui, ant design

---

✔ 프레임워크란 

> * 개발을 도와주는 도구 세트 
> * 기본적인 구조와 코드를 제공
> * 개발 속도를 높이고 효율적으로 일하기 위함

```bash
프레임워크는 우리가 필요한 것 외에도 되게 다양한 코드들을 지원하게 된다. 
그래서 우리가 불필요한 코드들을 우리 코드에 넣게 되고 오히려 프레임워크를 배움으로써 더 러닝 커브가 급격해질 수 있다. 
(러닝 커브 => 프레임워크를 배우는데 또 시간이 소요된다. )
```
